第一步：确认你需要哪些EPR注册

只要你向EU国家销售商品（含通过亚马逊/速卖通/Shopify等平台），就必须注册EPR。无最低销量门槛。

  1. 列出所有EU目标市场（德/法/西/意等）
  2. 判断每个产品涉及哪些EPR流：
    • 包装EPR — 几乎所有产品都需要（有包装就要注册）
    • WEEE(电子电器) — 电子产品、带电源的产品
    • 电池 — 含电池的产品（包括纽扣电池、锂电池）
    • 纺织 — 服装、家纺、鞋类（目前仅法国强制）
  3. 确认是否需要授权代表（非EU企业在德/法/西/意/波兰都需要）

PPWR 2026年8月12日起的新要求

  • 包装空隙率≤50% — 电商包裹（含气垫、泡沫等填充物）空隙不得超过50%
  • 有害物质限制 — 铅+镉+汞+六价铬总浓度 < 100mg/kg
  • PFAS限制 — 食品接触包装：单一PFAS < 25ppb，总量 < 250ppb
  • 可回收性 — 所有包装须符合EN 13430:2004标准
  • 合规声明 — 每种包装类型须完成评估并持有EU合规声明(DoC)
  • EPR注册 — 生产者须在每个销售国注册EPR
不合规后果：产品被平台下架 + 最高€200,000罚款 + 海关扣货 + 亚马逊自动启用代扣服务

紧急行动清单

现在就做（2026年Q2）

  1. 德国LUCID注册 — 免费，verpackungsregister.org，无门槛
  2. 法国EPR(UIN)注册 — 通过Citeo或Leko，每个品类需要单独UIN
  3. 西班牙Ecoembes注册 — 获取ENV号
  4. 上传EPR号到亚马逊 — Seller Central → 合规 → EPR合规
  5. 指定授权代表 — 非EU企业在每个销售国需要授权代表

2026年8月前必须完成

  1. 测试包装空隙率 — 确保 ≤ 50%（含填充物）
  2. 检测包装重金属含量 — 铅+镉+汞+六价铬 < 100mg/kg
  3. 准备合规声明(DoC) — 每种包装类型一份
  4. 意大利CONAI注册 — 需指定第三方代表
  5. 波兰BDO注册 — 通过华沙省政府办公室
  6. 含电池产品注册电池EPR

2027-2028年准备

  1. 包装数字标识(QR码) — 2027年起要求
  2. 统一EU包装标签 — 2028年8月起标准化象形图
  3. 纺织EPR — 关注各国转化进度（法国已生效，其他国家~2027）

各品类具体要求

包装EPR（所有卖家都涉及）

  • 德国：LUCID注册 → 与回收系统签约（Der Grüne Punkt等）→ 按材料种类和重量付费
  • 法国：2026年官费涨至€110/年（<10,000件），需要Triman标志
  • 西班牙：额外征收塑料包装税€0.45/kg不可回收塑料

电子电器WEEE

  • 覆盖6大类：大型家电、小型家电、IT/通讯、消费电子、照明、电动工具等
  • 非EU企业须指定授权代表
  • 需定期报告销售数量和重量

电池（2026年1月起强化）

  • 2026年1月起亚马逊要求上传电池EPR号（德法意西波等8国先行）
  • 2025年8月18日起非EU企业须有授权代表
  • 含电池产品（包括纽扣电池）都需要注册

纺织（法国已生效，其他国家2027+）

  • 法国：通过Refashion注册，覆盖服装/家纺/鞋类
  • 其他EU国家：2027年中各国需完成国内法转化
